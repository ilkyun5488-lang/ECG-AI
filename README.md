# 간호사를 위한 심전도(부정맥) 학습 앱 (PWA)

이 프로젝트는 **교육/학습 목적**의 부정맥 분류 앱 초안입니다.  
본 앱은 진단/치료 결정을 대체하지 않습니다.

## 실행 방법(가장 쉬운 방법)

- Windows에서 `index.html`을 더블클릭해서 열면 됩니다.
- 휴대폰(iPhone/갤럭시)에서 사용하려면, 나중에 이 폴더를 간단한 웹서버로 띄운 뒤(예: 학교/개인 PC), 링크로 접속하면 됩니다.

## 현재 구현 상태

- 첫 화면: **메뉴 선택(알고리즘으로 찾아가기 / 카메라 AI 판독(준비중))**
- 라우팅: 화면 전환을 위해 주소창의 `#`(해시) 기반 라우팅 사용

## 다음 단계(사용자가 임상 질문 제공)

- “알고리즘으로 찾아가기”에서 질문(예: 규칙성, QRS 폭, 빠름/느림, P파 유무)을 한 개씩 추가
- 답변 조합에 따라 후보 부정맥(1~2개) 제시
